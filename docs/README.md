# Math formulas
## Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²
- Triangle: S = ½ × b × h  (base × height / 2)
- Triangle (Heron's formula): S = √(p(p-a)(p-b)(p-c)), where p = (a+b+c)/2


## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a
- Triangle: P = a + b + c
